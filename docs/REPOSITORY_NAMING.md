## âš ï¸ **Pertimbangan Penting**

1. **Nama harus unik** di GitHub (tidak boleh sama dengan repository lain)
2. **Gunakan lowercase** dengan hyphen untuk pemisah kata (best practice)
3. **Hindari karakter khusus** (spasi, underscore, dll)
4. **Pertimbangkan panjang** - tidak terlalu pendek, tidak terlalu panjang
5. **Konsisten** dengan naming convention project lain Anda (jika ada)

---

## ğŸ” **Cek Ketersediaan Nama**

Sebelum membuat repository, cek apakah nama sudah tersedia:

1. Buka: `https://github.com/USERNAME/NAMA_REPOSITORY`
2. Jika muncul "404 Not Found", nama tersedia
3. Jika muncul halaman repository, nama sudah digunakan
